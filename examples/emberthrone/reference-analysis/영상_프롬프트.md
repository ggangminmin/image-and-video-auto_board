## 사용 메모
- **컷 길이 4~6초**로 짧게 뽑고, 후반 편집에서 비트에 맞춰 더 잘라 쓰는 걸 권장(원본이 평균 1.5초 컷).
- 우리 파이프라인 기본 = **무음(No-BGM)** → Kling `sound:"off"`. BGM/내레이션은 최종 조립에서 합성.
- 각 영상은 `이미지_프롬프트.md` 의 동일 번호 키프레임을 start_image 로.
- 리뷰·재생성은 `/mcp-review-console` 콘솔로 폴더 띄워서 컷별 수정/영상전환.
