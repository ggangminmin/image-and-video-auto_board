# EMBERTHRONE — 편집 시퀀스 35컷 (재배열 + 추가)

> 30초 3D 시네마틱 게임 트레일러. 단일 영웅의 *rise*(몰락→각성→승리).
> 레퍼런스 **Coway 릴스(DZcLoyQS4TM) 0.3초 분석**의 리듬·카메라무브(CM01~CM15)를 EMBERTHRONE 세계관에 이식.
